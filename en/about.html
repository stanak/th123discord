About This Page¶

What Type of Server This is?¶

In this server, we enjoy a wide range of activities via th123 games, th123-related events, casual chats, even non-th123 games, and so on. The main goal of this server is to provide a stable environment for communication between th123 players available for a long time. We welcome any type of th123 player: active players, beginners, past players, and newcomers who do not start yet but consider to start th123.

Server Entrance¶

Click Enter this server and find the entrance of the server.

What is Discord?¶

Discord is a light-weight, modern, and gaming-use focused communication tool. It supports a wide range of operating systems and mobile devices. Japanese Discord official says “The time for Skype and TeamSpeak is over!” with a provocative mood.

For details, search google about Discord yourself.

Notes¶

For a comfortable time for everyone in the server, the server manager may give warnings, cautions, or even BANs from the server to the inappropriate behaviors. In case you are troubled with such an attendee, please feel free to contact the manager about it. We consider the following items as inappropriate in the server:

  • Activities contrary to public order or morality

  • Any kinds of harassments

  • Any kinds of cracking trials

  • Other activities contrary to public interests

Getting Contact¶

When you need to contact the manager outside of #servermeta, send a DM to anyone assigned a MANAGER role.
